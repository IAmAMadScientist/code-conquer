# Contributing to Code & Conquer

Thanks for your interest in contributing!

## Quick Start
1. Fork the repo and create a branch from `main`
2. Install dependencies:
   - Frontend: `cd frontend && npm install`
   - Backend: `cd server && ./mvnw spring-boot:run`
3. Run frontend:
   - `cd frontend && npm run dev`

## Development Guidelines
- Keep PRs small and focused
- Use clear commit messages (what + why)
- Add/update documentation if behavior changes
- Prefer mobile-first UI changes (test on real devices if possible)

## Code Style
- Frontend: follow ESLint (run `npm run lint` if available)
- Backend: keep formatting consistent with existing code

## Submitting a Pull Request
- Describe the change and motivation
- Link related issues
- Add screenshots for UI changes (mobile + desktop if relevant)

## Reporting Bugs / Requesting Features
Please use the GitHub issue templates.
